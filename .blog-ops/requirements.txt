# iMagic Blog - 工具脚本依赖

## 依赖列表

- **Pillow** >= 9.0.0 —— 图片缩放、格式转换（process_image.py）
- **requests** >= 2.25.0 —— HTTP 请求（ai_client.py, blog_runner.py, d1_client.py, search_client.py）

## 安装方法

```bash
pip install -r requirements.txt
```

## 说明

- `create_post.py` —— 仅使用标准库（os, argparse, re, datetime）
- `lint_post.py` —— 仅使用标准库（os, argparse, re, datetime, sys）
